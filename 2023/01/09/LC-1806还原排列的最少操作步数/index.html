LC-1806还原排列的最少操作步数

本文最后更新于 2024年2月10日

题目链接 | 第 234 场单周赛 | Q2 | 难度:1491(+400)(难度取决于数据量)

题目描述

给你一个偶数 n​​​​​​ ，已知存在一个长度为 n 的排列 perm ，其中 perm[i] == i​（下标 从 0 开始 计数）。

一步操作中，你将创建一个新数组 arr ，对于每个 i ：

  • 如果 i % 2 == 0 ，那么 arr[i] = perm[i / 2]
  • 如果 i % 2 == 1 ，那么 arr[i] = perm[n / 2 + (i - 1) / 2]

然后将 arr​​ 赋值​​给 perm 。

要想使 perm 回到排列初始值，至少需要执行多少步操作？返回最小的 非零 操作步数。

 

示例 1：

输入：n = 2
输出：1
解释：最初，perm = [0,1]
第 1 步操作后，perm = [0,1]
所以，仅需执行 1 步操作

示例 2：

输入：n = 4
输出：2
解释：最初，perm = [0,1,2,3]
第 1 步操作后，perm = [0,2,1,3]
第 2 步操作后，perm = [0,1,2,3]
所以，仅需执行 2 步操作

示例 3：

输入：n = 6
输出：4

 

提示：

  • 2 <= n <= 1000
  • 1800分难度:2 <= n <= 1e5
  • n​​​​​​ 是一个偶数

题目解析

这道题的数学解法是直接求 欧拉函数(复杂度 $O(n)$), 可以看 官方题解, 这里给出使用 并查集 求环的解法, 复杂度 $O(nlgn)$.

这道题其实就是求出所有环的大小, 求他们的最小公倍数 (lcm). 所以用 并查集 求环和其大小.
环其实就是一个排列的 置换, 也就是要让所有的 置换 都回归到自己的位置, 一个 置换 的一个 循环 就是 环 的大小.
